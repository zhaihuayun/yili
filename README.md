
周易之理

或许这是上个文明传递下来的，通过河图洛书留下来的，
也可能是中华文明的发明，谁发明的不重要了，
关键是传承的使命，到我们了。

易理，是天地人，运行的模型。
在天为象，为表面，易理，说的是背景后的动力。
以及演进变化过程。

从河图洛书，抽象出甲子八卦，就是得出时间与空间的表示，
这不过就是把1进行2 4 6 8 分的PID控制。
爻，不过是对应周期的时空单位。
周易，也是就为了以结果来推导控制过程的控制理论。
只是它是以果决行，以结果倒推阶段。

推的是什么？
是阴阳的张力与延伸。
是正负极向导通时间单位后，对应的空间力量。
升频，则导通时间长。

正向导通时间长，则产生空间的力量，
体现为对等的张力回归与延伸。

空间的期待，不是是半步崩拳的自然释放。
